# SkillsBank

当前功能：

- 在侧边栏显示由 `skills-config.json` 配置的选项列表
- 支持 `menu` 类型（可展开）
- 支持 `skill` 类型（可点击）
- 点击 `skill` 仅提示信息，不执行实际逻辑
- 侧边栏底部提供 `修改配置` 按钮，点击后打开 `skills-config.json`

## 开发调试启动方式

1. 安装依赖：`npm install`
2. 编译：`npm run compile`
3. 在 Cursor/VS Code 中按 `F5` 启动扩展调试
4. 点击左侧活动栏 `SkillsBank` 图标查看列表
